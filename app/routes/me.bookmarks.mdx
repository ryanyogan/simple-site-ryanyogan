# My Favorite Languages and Paradigms: Past, Present, and Future

Welcome to my curated collection of programming languages and frameworks that have shaped my journey as a developer. This list represents the tools I've loved, the ones I'm currently excited about, and those I'm looking forward to exploring.

## Programming Languages

- TypeScript: A typed superset of JavaScript that compiles to plain JavaScript. [Learn More](https://www.typescriptlang.org/)
- Elixir: A dynamic, functional language designed for building scalable and maintainable applications. [Learn More](https://elixir-lang.org/)
- Rust: A language empowering everyone to build reliable and efficient software. [Learn More](https://www.rust-lang.org/)
- Go: An open source programming language that makes it easy to build simple, reliable, and efficient software. [Learn More](https://golang.org/)
- Python: A clear and powerful object-oriented programming language, comparable to Perl, Ruby, Scheme, or Java. [Learn More](https://www.python.org/)
- JavaScript: JavaScript is a programming language that conforms to the ECMAScript specification. [Learn More](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- Ruby: A dynamic, open source programming language with a focus on simplicity and productivity. [Learn More](https://www.ruby-lang.org/en/)
- C#: A simple, general-purpose, object-oriented programming language for the .NET platform. [Learn More](https://docs.microsoft.com/en-us/dotnet/csharp/)
- Kotlin: A cross-platform, statically typed, general-purpose programming language with type inference. [Learn More](https://kotlinlang.org/)
- Swift: A powerful and intuitive programming language for macOS, iOS, watchOS [Learn More](https://developer.apple.com/swift/)
- Dart: A client-optimized language for fast apps on any platform. [Learn More](https://dart.dev/)
- Lua: A powerful, efficient, lightweight, embeddable scripting language. [Learn More](https://www.lua.org/)
- Clojure: A dynamic, general-purpose programming language, combining the approachability and interactive development of a scripting language with an efficient and robust infrastructure for multithreaded programming. [Learn More](https://clojure.org/)
- Erlang: A general-purpose, concurrent, functional programming language, and a garbage-collected runtime system. [Learn More](https://www.erlang.org/)
- PHP: PHP 3... This was the first language I used that brought a paycheck in. Always and forever! [Learn More](https://www.php.net/)

- Next.JS: The React Framework for Production. [Learn More](https://nextjs.org/)
- Phoenix: A productive web framework that does not compromise speed and maintainability. [Learn More](https://www.phoenixframework.org/)
- Node.js: A JavaScript runtime built on Chrome's V8 JavaScript engine. [Learn More](https://nodejs.org/)
- Django: A high-level Python Web framework that encourages rapid development and clean, pragmatic design. [Learn More](https://www.djangoproject.com/)
- React: A JavaScript library for building user interfaces. [Learn More](https://reactjs.org/)
- Angular: A platform and framework for building single-page client applications using HTML and TypeScript. [Learn More](https://angular.io/)
- Svelte: A new way to build web applications. [Learn More](https://svelte.dev/)
- Remix: A full-stack web framework for the modern web. [Learn More](https://remix.run/)
- Tokio: A runtime for writing reliable, asynchronous, and slim applications with the Rust programming language. [Learn More](https://tokio.rs/)
- iOS: A mobile operating system created and developed by Apple Inc. exclusively for its hardware. [Learn More](https://developer.apple.com/ios/)
- Android: A mobile operating system developed by Google. [Learn More](https://developer.android.com/)
- .NET: A free, cross-platform, open-source developer platform for building many different types of applications. [Learn More](https://dotnet.microsoft.com/)
- Watermill.io: A Go library for working efficiently with message streams. [Learn More](https://watermill.io/)
- Kubernetes: An open-source platform designed to automate deploying, scaling, and operating application containers. [Learn More](https://kubernetes.io/)
- Flutter: Google's UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase. [Learn More](https://flutter.dev/)
- React Native: A framework for building native apps using React and JavaScript. [Learn More](https://reactnative.dev/)
- Expo: A framework and a platform for universal React applications. [Learn More](https://expo.dev/)

This list is ever-evolving, just like the world of technology. Stay curious, keep learning, and happy coding!
